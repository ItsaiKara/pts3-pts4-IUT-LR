EmulateGPS module¶

class EmulateGPS.EmulateGPS[source]¶

Bases: object

This class emulate a route between UIT of La Rochelle and Natuition’s office.

This route is composed by one thousand point. One point is defiend by a latitude and a longitude.

This class has to be used to test inserting potentiometer results in database whitout use the robot.

This method initialize the route parameters :

  • start point of route,

  • end point of route.

These values can be modified to create other routes (it’s useless).

Authors of this class : CHENUAUD Emmanuel.
