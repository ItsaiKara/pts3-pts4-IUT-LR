SensorProcessing module¶

class SensorProcessing.SensorProcessing(pathConfig, channel)[source]¶

Bases: object

Class managing the sensor’s class and the database’s class.

Inits SensorProcessing with path of the configuration file.

Create an instance of AnalogDigitalConverter, Database and thread.

And if configuration file doesn’t exist he create it.

Parameters

  • pathConfig – Path of the configuration file.

  • channel – Channel number where the potentiometer is connected [0|1|2].
